# Boilerplate Web Moderna

## Descripción del proyecto
Este proyecto es un **boilerplate web moderno** diseñado para servir como base para proyectos profesionales. Está construido con **HTML semántico** y **CSS Flexbox**, inspirado en el diseño de Apple.com. Incluye una estructura adaptable, componentes reutilizables y diseño responsive para garantizar compatibilidad con distintos dispositivos.

---

## Funcionalidades
- **Diseño moderno y minimalista:** Estructura limpia y clara, ideal para proyectos profesionales.
- **Responsive y adaptable:** Garantiza un diseño óptimo en móviles, tablets y escritorio.
- **Flexbox:** Organización flexible de contenidos para crear layouts dinámicos.
- **Secciones predefinidas:**
  - Header con navegación principal y logo.
  - Hero section con título, descripción y botón de acción.
  - Grid de artículos con imagen, título, descripción y enlace.
  - Sidebar con widgets de búsqueda e información adicional.
  - Footer con derechos de autor y redes sociales.
- **Accesibilidad:** Uso de `aria-label` y etiquetas semánticas para mejorar la navegación.

---

## Tecnologías utilizadas
- **HTML5:** Estructura semántica del sitio.
- **CSS3 (Flexbox):** Diseño adaptable y moderno.
- **Google Fonts:** Fuente Inter para tipografía profesional.
- **Font Awesome:** Iconos para logo y redes sociales.
- **JavaScript (opcional):** Para futuras interacciones dinámicas.
